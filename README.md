# 無人機方向訓練工具

一個互動式網頁應用程式，訓練無人機操作員在不同朝向下的方向控制能力。

## 🎮 遊戲玩法

使用方向鍵操控無人機飛向目標寶石。無人機會隨機面向四個方向（前、後、左、右），每個方向的控制邏輯都不同：

- **機尾朝你（背面）**: 直觀控制 - 上下左右對應螢幕方向
- **機鼻朝你（正面）**: 反向控制 - 所有方向相反
- **側面朝你**: 90度旋轉控制 - 需要轉換思考

## 🎯 目標

訓練肌肉記憶，在任何朝向下都能快速正確地操控無人機。

## 🚀 線上遊玩

訪問: `https://[你的GitHub用戶名].github.io/drone-practice/`

## 📁 專案結構

```
├── index.html       # 主頁面
├── script.js        # 遊戲邏輯
├── style.css        # 樣式表

├── drone_front.png  # 機鼻朝向圖片

└── drone_right.png  # 右側圖片
```

## 🛠️ 本地執行

1. 下載所有檔案到同一資料夾
2. 用瀏覽器開啟 `index.html`
3. 開始練習！

## 📝 控制說明

詳細的方向控制邏輯請參考遊戲內的提示系統。遊戲會在失敗後提供正確按鍵的提示。
